```markdown
- 프로젝트 진행 상황 추적

### 2순위: TodoWrite (제한적 사용)
- 단순한 체크리스트
- 즉시 완료 가능한 작업들

### 금지: 임의 도구 변경
- 진행 중인 작업의 도구 변경 금지
- 사용자 명시적 요청 없이 도구 전환 금지
```

## 📋 **모델별 특성 이해**

### **Opus 4.1 특성**
- 깊은 사고와 복잡한 추론
- 체계적이고 신중한 접근
- 상세한 분석과 비교

### **Sonnet 4 특성**  
- 빠른 실행과 효율성
- 간결하고 직접적 접근
- 실용적 해결책 중심

### **공통 유지 사항**
- **CLAUDE.md 지침 절대 준수**
- **기존 도구와 방법론 유지**
- **포터블 철학 일관성**
- **맥락 중심 개발 방식**
